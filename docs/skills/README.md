# Skills

仓库提供可分发到宿主项目的 skill 源文件。

## 内容

- `skills/casdoor-next-auth-kit/SKILL.md`
- `skills/casdoor-next-auth-kit/references/`

## 安装

在仓库根目录运行辅助脚本：

```bash
node scripts/install-skill.mjs /path/to/host-project
```
